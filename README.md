# utility_C

### Quick start 

```bash
# run demo c program
$ gcc -o helloworld helloworld.c
$ ./helloworld

# run makefile 
$ make 
# cmd    		| help
# --------------------------------------------------------------------------------------
# help  		| show help table 
# helloworld  	| dev 
# build		| dev 
# clean		| dev 
# install		| dev 

$make build 
# gcc -o for_loop for_loop.c
# gcc -o while while.c
# gcc -o array array.c

```
